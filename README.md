Event Enquiry Website

A modern and responsive Event Enquiry Website that allows users to explore events and send enquiries for bookings, registrations, or event-related information.

🚀 Features
🎉 Browse upcoming events
📩 Send event enquiries through contact forms
📅 Event details page
🔍 Search and filter events
📱 Fully responsive design
🔐 Form validation
⚡ Fast and user-friendly interface
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Node.js / Express (optional)
Database: MongoDB / MySQL (optional)
Version Control: Git & GitHub
